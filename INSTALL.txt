======================================================================
Installation
======================================================================

First, unpack the zip-file and copy or link the rSPM folder to the spm5/toolbox directory. You can start the toolbox by choosing "rSPM" in the "Toolboxes" selector. Alternatively you can use the "TASKS" button (right in the Graphics window): select TASKS | Tools | rSPM.
